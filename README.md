# Web Tools - 网页工具集

一个精心设计的在线网页工具集合，提供实用、高效的浏览器端工具，无需安装，即开即用。

## 📖 项目简介

Web Tools 是一个轻量级的纯前端网页工具集，致力于为用户提供简洁、优雅的在线工具体验。所有工具均运行在浏览器端，无需后端服务，保护用户隐私，加载迅速，支持离线使用。

## ✨ 核心特性

- 🚀 **纯前端实现** - 无需服务器，完全在浏览器中运行
- 🔒 **隐私安全** - 所有数据处理在本地完成，不上传任何信息
- 📱 **响应式设计** - 完美适配桌面端和移动端设备
- ⚡ **快速加载** - 轻量级设计，秒级打开
- 🎨 **优雅界面** - 现代化 UI 设计，流畅的交互体验
- 💾 **离线可用** - 支持 PWA，可添加到主屏幕离线使用

## 🚀 快速开始

### 本地运行

1. **克隆项目**
```bash
git clone <repository-url>
cd web_tools
```

2. **直接打开**
   - 直接用浏览器打开 `index.html` 即可访问工具导航页
   - 或分别打开各工具目录下的 `index.html`

3. **使用本地服务器（推荐）**
```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js 的 http-server
npx http-server -p 8080
```

然后在浏览器中访问 `http://localhost:8080`

### 在线访问

直接访问部署后的网站即可使用所有工具。

### 🔧 修改域名配置

如需修改网站域名，只需 **1 步**：

**编辑配置文件** `config.js`：
```javascript
const SITE_CONFIG = {
    domain: "https://your-new-domain.com",  // 修改这里
    // ... 其他配置
};
```

**所有页面的SEO标签会自动使用新域名！**

> **注意**：sitemap.xml 和 robots.txt 是静态文件，如需修改请手动全局替换其中的域名。
